# 软件条目模板

```yaml
- id:
  name:
  category:
  vendor:
  status: active
  homepage:
  download_page:
  official_source:
  install:
  platforms:
    - macOS
    - Windows
    - Linux
  pricing:
  source_type: official
  tags:
    -
  last_checked: YYYY-MM-DD
  notes:
```

填写要求：

- `download_page` 不要写第三方下载站。
- `install` 只写官方文档明确给出的命令。
- `notes` 写限制、风险、授权、平台要求，不写主观推荐。
