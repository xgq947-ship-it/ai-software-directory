# 开发工作台工具

这类工具不是全部都属于 AI 软件，但会直接影响 AI 开发、远程维护、截图记录和多 Agent 工作流。

| 名称 | 官方入口 | 类型 | 说明 |
| --- | --- | --- | --- |
| UU 远程 | https://uuyc.163.com/download/ | 远程控制 | 网易官方远程控制工具，支持 Windows、macOS、iOS、Android、Android TV |
| Snipaste | https://www.snipaste.com/download.html | 截图贴图 | 截图、标注、贴图、取色和视觉资料整理 |
| cmux | https://cmux.com/ | 开发终端 | macOS 终端，适合并行运行 Claude Code、Codex、OpenCode、Gemini CLI、Aider 等 Agent |
| CC Switch | https://ccswitch.io/zh/ | AI CLI 管理 | 统一管理 Claude Code、Codex、Gemini CLI、OpenCode、OpenClaw、Hermes Agent 等 CLI 工作流 |

维护规则：

- 远程控制和截图工具只收官方入口，不收第三方下载站。
- cmux 和 CC Switch 优先收官网与 GitHub Release，不固定写某个版本安装包。
- 如果用户写 `ccwitch`，默认按 `CC Switch / cc-switch` 处理，除非另有链接。
