# AI 客户端

闭源大模型通常不提供权重下载。这里记录官方客户端、平台入口和应用商店入口。

| 名称 | 官方入口 | 说明 |
| --- | --- | --- |
| ChatGPT Desktop | https://chatgpt.com/features/desktop/ | OpenAI 官方桌面客户端 |
| Claude Desktop | https://claude.ai/download | Anthropic 官方桌面客户端 |
| Gemini for macOS | https://gemini.google/mac/ | Google 官方 macOS 客户端 |
| Perplexity | https://www.perplexity.ai/platforms | Perplexity 官方多平台入口 |

不要把第三方封装版、搬运版、网盘版写进主数据。
