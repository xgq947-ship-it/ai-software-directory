# Android 自动化

这类工具主要收 APK 下载入口。优先记录官网、官方下载页或 GitHub Release。

| 名称 | 官方入口 | APK 入口 | 说明 |
| --- | --- | --- | --- |
| Hamibot | https://hamibot.com/ | https://hamibot.com/download/ | 安卓自动化工具，官网提供稳定版 APK |
| AutoX.js | https://github.com/autox-community/AutoX | https://github.com/autox-community/AutoX/releases | Android 自动化工具，普通用户优先下载 universal APK |

维护规则：

- APK 直链只有官方稳定别名或 GitHub Release 资产才记录。
- AutoX.js 这类多架构 APK，普通用户优先选 `universal`。
- 不收第三方下载站、网盘搬运包、修改版、破解包。
