# 本地模型运行工具

这类工具用于下载、管理和运行本地模型。优先记录官方下载页。

| 名称 | 官方入口 | 说明 |
| --- | --- | --- |
| Ollama | https://ollama.com/download | CLI / 桌面本地模型运行工具 |
| LM Studio | https://lmstudio.ai/download | 图形化本地模型工具 |
| Jan | https://www.jan.ai/download | 开源桌面 AI 客户端 |
| AnythingLLM | https://anythingllm.com/desktop | 本地 RAG、文档问答、Agent |

建议记录系统要求、GPU/内存要求和模型格式，但不要把大模型文件直接放进仓库。
