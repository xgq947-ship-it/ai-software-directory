# macOS 实用工具

这类工具主要面向普通用户，优先记录官网、GitHub Release、Homebrew Cask 或官方应用商店入口。

| 名称 | 官方入口 | 安装入口 | 说明 |
| --- | --- | --- | --- |
| Vorssaint | https://vorssaint.com/ | https://github.com/vorssaint/vorssaint-utils/releases | macOS 菜单栏工具箱，支持 Homebrew Cask 和 DMG 安装 |

维护规则：

- 只收官方 Release、官网、Homebrew Cask 或应用商店入口。
- 如果有 Homebrew Cask，记录安装命令，但 README 优先展示普通用户能打开的下载页。
- 不收第三方下载站、网盘搬运包、破解包。
